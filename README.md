# IA_Metaheuristicas

Os arquivos das meta-heuristicas podem ser executados separadamente para testes individuais. Eles são:
1) gradient_descent.py
2) tabu_seach.py
3) grasp.py

Os arquivos principais são:
1) treino.py, que roda o treino para o ajuste de parâmetros;
2) teste.py, que efetua o teste comparativo das metaheurísticas.
